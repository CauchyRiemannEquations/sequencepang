# 🍈 수열팡 (Sequence Pang) - 타임어택 수학 퍼즐 웹 게임 (v1.0)

고등학생들이 수열(등차수열, 등비수열)의 개념을 직관적이고 흥미진진하게 학습할 수 있도록 개발된 **웹 기반 타임어택 수학 퍼즐 게임**입니다.

복잡한 가입 절차나 네트워크 세팅 없이 브라우저에서 실행 즉시 60초 동안 점수를 겨루는 싱글 플레이 게임으로, 수업 시간이나 자기주도적 학습 교구로 가볍고 재미있게 활용할 수 있도록 최적화되어 있습니다.

---

## ✨ 핵심 기능 (Features)

1. **상큼한 메론 테마 퍼즐 보드**
   - 6x6 그리드 위에 아기자기하고 예쁜 **배경 투명 메론 일러스트 타일**이 채워지며, 타일마다 1부터 10까지의 숫자가 선명하게 조화를 이룹니다.
   - 격자판 배경을 완전 투명화하여 탐스럽고 둥근 메론들이 화면 위에 입체감 있게 둥둥 떠 있는 쾌적하고 감각적인 화면을 선사합니다.
   
2. **직관적인 드래그 & 스와이프 인터랙션**
   - 마우스 드래그 또는 모바일/태블릿 터치 스와이프로 3개 이상의 타일을 연속해서 이을 수 있습니다.
   - 드래그 도중 실수했을 때 뒤로 되돌아가면 방금 선택한 타일이 취소되는 **실시간 되돌리기(Backtracking)** 기능을 완벽히 지원합니다.
   - 드래그 궤적과 광채 피드백 역시 메론 형상에 맞추어 **둥근 원형(radius 50%)**으로 연출되어 시각적 몰입감이 뛰어납니다.

3. **등차·등비수열 판정 엔진**
   - 연결한 수들이 **등차수열**($d \neq 0$) 또는 **등비수열**($r \neq 1$)을 이루면 점수를 획득하고 타일이 팡팡 터집니다.
   - 연속해서 맞출수록 콤보 점수가 누적되어 더 큰 고득점을 노릴 수 있습니다.

4. **타임어택 & 콤보 시스템**
   - 제한 시간 60초에서 시작합니다.
   - 수열 매칭 성공 시 **보너스 시간**(+콤보 가중치 추가)을 얻습니다.
   - 오답 매칭 시 시간 단축 패널티는 **-3.0초**로 조율되어 있어 긴장감 넘치는 템포를 유지합니다.

5. **눈부심 방지 포레스트 톤 & 모바일 반응형 디자인**
   - 어두운 교실이나 야간 자습 시 학생들의 눈 피로도를 낮추기 위해 **차분한 딥 포레스트 그린(#121e15)**과 **미색 아이보리(#f5f9f4)** 테마를 기본으로 적용했습니다.
   - 모바일 세로 모드 및 PC/태블릿 가로 모드 전체에서 6x6 보드가 화면 크기에 맞게 피팅되는 유연한 반응형 레이아웃을 제공합니다.

6. **상큼하고 정갈한 Jua 폰트 감성**
   - 밋밋하고 딱딱한 기본 폰트 대신 둥글둥글하고 친근한 인기 구글 폰트인 **`Jua` (주아체)**를 메인 폰트로 전격 도입했습니다.
   - 글꼴의 자연스러운 정갈함을 살리기 위해 찌그러지는 강제 볼드(Synthetic Bold) 처리를 일체 해제하여 모바일 정식 스토어 게임처럼 쾌적하고 깔끔하게 렌더링됩니다.

## 📂 저장소 파일 구조 (Repository Files)

GitHub 저장소(Repository)에 업로드하여 배포할 때 필요한 파일은 단 **두 개**입니다!

```bash
├── index.html   # 게임의 전체 UI 구조, CSS 스타일 및 자바스크립트 엔진 (단일 파일 포맷)
├── melon.png    # 퍼즐 타일에 입혀지는 배경 투명 고화질 메론 이미지 리소스
└── README.md    # 현재 보고 계시는 프로젝트 안내 문서
```

> [!IMPORTANT]
> 메론 타일 그래픽을 온전히 표현하기 위해 `index.html`과 `melon.png`가 **반드시 동일한 폴더 위치**에 업로드되어야 합니다!

## 🚀 GitHub Pages를 이용한 1분 온라인 무료 배포 방법

GitHub Pages를 활용하면 전 세계 누구든 접속할 수 있는 무료 게임 링크(`.github.io`)를 개설할 수 있습니다.

### 1단계: GitHub 저장소 만들기
1. 본인의 [GitHub](https://github.com/) 계정으로 로그인한 뒤, 우측 상단의 **`New`** 버튼을 클릭하여 새 저장소(Repository)를 생성합니다.
2. 저장소 이름(Repository Name)을 적당히 입력합니다 (예: `sequence-pang`).
3. 공개 범위는 **`Public`**으로 설정해야 무료 배포가 가능합니다.
4. **`Create repository`**를 누릅니다.

### 2단계: 파일 업로드하기
1. 생성된 저장소 페이지에서 **`uploading an existing file`** 링크를 클릭합니다.
2. 로컬 컴퓨터의 `sequence-pang` 폴더에서 **`index.html`**과 **`melon.png`** 파일을 한꺼번에 드래그 앤 드롭하여 업로드창에 올립니다.
3. 화면 하단의 녹색 **`Commit changes`** 버튼을 눌러 저장을 완료합니다.

### 3단계: Pages 배포 설정 켜기
1. 저장소 상단의 톱니바퀴 아이콘인 **`Settings`** 메뉴로 진입합니다.
2. 좌측 사이드바 메뉴 중 **`Pages`** 메뉴를 클릭합니다.
3. **Build and deployment** 항목의 **Branch** 영역에서 `None`으로 되어 있는 드롭다운을 클릭하여 **`main`** (또는 `master`) 브라우저로 설정하고 폴더는 `/ (root)` 상태 그대로 우측의 **`Save`** 버튼을 누릅니다.

### 4단계: 무료 주소 접속하기
1. 설정 후 1~2분 정도 기다리신 뒤 새로고침을 하거나 페이지 상단에 노출되는 **"Your site is live at..."** 링크를 클릭합니다.
2. 보통 다음과 같은 포맷의 무료 도메인 주소가 활성화됩니다:
   `https://[본인깃허브아이디].github.io/sequence-pang/`
3. 이 링크를 스마트폰이나 학교 스마트 보드에 접속하여 바로 싱글 타임어택 경쟁을 즐기면 됩니다!

## 🛠️ 기술적 참고 사항 (Tech Stack)
- **Frontend:** Vanilla HTML5, Pure CSS3, Modern JS ES6 (Flexbox & Grid responsive support)
- **Typography:** Google Fonts `Jua` (Korean Main) & `Outfit` (Numeric Accent)
- **Aesthetics:** HSL 딥 포레스트 다크 모드, 완전 투명 메론 일러스트, 50% 원형 백드롭 광채 인터랙션
