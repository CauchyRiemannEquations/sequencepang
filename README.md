# 🍈 수열팡 (Sequence Pang) - 실시간 타임어택 수학 퍼즐 웹 게임

고등학생들이 수열(등차수열, 등비수열)의 개념을 직관적이고 흥미진진하게 학습할 수 있도록, 기존 '사과게임'에서 착안하여 개발된 **웹 기반 실시간 타임어택 수학 퍼즐 게임**입니다.

싱글 플레이뿐만 아니라, **동일한 방 코드를 입력한 학생들끼리 실시간으로 점수 경쟁을 벌일 수 있는 리더보드 기능**이 탑재되어 있어 교실 수업이나 이벤트 교구로 활용하기에 최적화되어 있습니다.

---

## ✨ 핵심 기능 (Features)

1. **상큼한 메론 테마 퍼즐 보드**
   - 6x6 그리드 위에 아기자기한 메론 일러스트 타일이 채워지며, 타일마다 1부터 10까지의 숫자가 선명하게 중앙 정렬되어 있습니다.
   
2. **직관적인 드래그 & 스와이프 인터랙션**
   - 마우스 드래그 또는 모바일/태블릿 터치 스와이프로 3개 이상의 타일을 연속해서 이을 수 있습니다.
   - 드래그 도중 뒤로 되돌아가면 방금 선택한 타일이 취소되는 **실시간 되돌리기(Backtracking)** 기능을 지원합니다.

3. **등차·등비수열 판정 엔진**
   - 연결한 수들이 **등차수열**($d \neq 0$) 또는 **등비수열**($r \neq 1$)을 이루면 점수를 획득하고 타일이 터집니다.
   - 콤보가 쌓일수록 획득 점수가 가속화됩니다.

4. **타임어택 & 콤보 시스템**
   - 제한 시간 60초에서 시작합니다.
   - 수열 매칭 성공 시 **시간 추가 보너스**(+콤보 가중치)를 얻습니다.
   - 오답 매칭 시 **-3초 시간 단축 패널티**가 부여됩니다.

5. **실시간 온라인 멀티플레이 리더보드**
   - 별도 가입 절차 없이 **방 코드(Room Code)**를 입력하는 것만으로 Firebase Realtime Database 클라우드에 즉시 연결됩니다.
   - 같은 방 코드로 접속한 전체 플레이어의 점수가 실시간으로 우측 랭킹보드에 0.1초마다 동기화되어 짜릿한 실시간 점수 경쟁이 가능합니다.

6. **눈부심 방지 포레스트 톤 & 모바일 반응형 디자인**
   - 교실의 어두운 빔프로젝터나 야간 학습 시 눈이 피로하지 않도록 차분한 딥그린(#121e15)과 미색 아이보리(#f5f9f4) 테마를 도입했습니다.
   - 모바일 세로 모드와 PC/태블릿 가로 모드에 각각 최적화된 유연한 반응형 레이아웃을 제공합니다.

## 📂 저장소 파일 구조 (Repository Files)

GitHub 저장소(Repository)에 배포할 때 필요한 파일은 단 **두 개**입니다!

```bash
├── index.html   # 게임의 전체 UI 구조, CSS 스타일 및 자바스크립트 엔진 (단일 파일 포맷)
├── melon.png    # 퍼즐 타일에 입혀지는 고화질 메론 벡터 이미지 리소스
└── README.md    # 현재 보고 계시는 프로젝트 안내 문서
```

> [!WARNING]
> 메론 타일 그래픽을 온전히 표현하기 위해 `index.html`과 `melon.png`가 **반드시 동일한 폴더 위치**에 업로드되어야 합니다!

## 🚀 GitHub Pages를 이용한 1분 온라인 무료 배포 방법

GitHub Pages를 활용하면 전 세계 누구든 접속할 수 있는 무료 게임 링크(`.github.io`)를 개설할 수 있습니다.

### 1단계: GitHub 저장소 만들기
1. 본인의 [GitHub](https://github.com/) 계정으로 로그인한 뒤, 우측 상단의 **`New`** 버튼을 클릭하여 새 저장소(Repository)를 생성합니다.
2. 저장소 이름(Repository Name)을 적당히 입력합니다 (예: `sequence-pang`).
3. 공개 범위는 **`Public`**으로 설정해야 무료 배포가 가능합니다.
4. **`Create repository`**를 누릅니다.

### 2단계: 파일 업로드하기
1. 생성된 저장소 페이지에서 **`uploading an existing file`** 링크를 클릭합니다.
2. 로컬 컴퓨터의 `C:\Users\USER\.gemini\antigravity\scratch\sequence-pang` 폴더에서 **`index.html`**과 **`melon.png`** 파일을 한꺼번에 드래그 앤 드롭하여 업로드창에 올립니다.
3. 화면 하단의 녹색 **`Commit changes`** 버튼을 눌러 저장을 완료합니다.

### 3단계: Pages 배포 설정 켜기
1. 저장소 상단의 톱니바퀴 아이콘인 **`Settings`** 메뉴로 진입합니다.
2. 좌측 사이드바 메뉴 중 **`Pages`** 메뉴를 클릭합니다.
3. **Build and deployment** 항목의 **Branch** 영역에서 `None`으로 되어 있는 드롭다운을 클릭하여 **`main`** (또는 `master`) 브라우저로 설정하고 폴더는 `/ (root)` 상태 그대로 우측의 **`Save`** 버튼을 누릅니다.

### 4단계: 무료 주소 접속하기
1. 설정 후 1~2분 정도 기다리신 뒤 새로고침을 하거나 페이지 상단에 노출되는 **"Your site is live at..."** 링크를 클릭합니다.
2. 보통 다음과 같은 포맷의 무료 도메인 주소가 활성화됩니다:
   `https://[본인깃허브아이디].github.io/sequence-pang/`
3. 이 링크를 학생들에게 스마트폰 메시지나 스마트 보드로 공유하면 끝입니다!

## 🎮 멀티플레이 랭킹 동기화 확인하는 법

온라인 배포 완료 후 또는 로컬 상태에서도 다음 단계로 완벽한 실시간 연동을 체감해 보실 수 있습니다.

1. 인터넷 브라우저로 배포된 주소에 접속합니다.
2. 닉네임에 `교사_홍길동`을 입력하고, 방 코드에 `math101`을 입력한 뒤 **🍈 메론팡 시작하기**를 클릭합니다.
3. 또 하나의 브라우저 탭(또는 시크릿 창, 스마트폰)을 띄우고 같은 주소로 접속합니다.
4. 닉네임에 `학생_김철수`, 방 코드에 똑같이 `math101`을 입력하고 입장합니다.
5. 우측 **실시간 랭킹보드**에 `교사_홍길동`과 `학생_김철수`가 동시에 표시되며, 한쪽에서 퍼즐을 맞출 때마다 등수가 실시간으로 오르내리는 진풍경을 확인하실 수 있습니다!

## 🛠️ 기술적 참고 사항 (Tech Stack)
- **Frontend:** Vanilla HTML5, Pure CSS3, Modern JS ES6 (Flexbox & Grid responsive support)
- **Database:** Firebase Database SDK Compat (Version 10.8.0 Compat CDN)
- **Database Endpoint:** 내장된 무료 공개 실시간 데이터베이스 노드가 연결되어 있어 초기 세팅 없이 즉시 가동됩니다. 필요시 대기화면 하단의 `Firebase 커스텀 설정`을 통해 개인 소유의 Realtime Database Config JSON을 등록해 독립적으로 관리하실 수도 있습니다.
